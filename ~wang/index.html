程序与文档

下列所有程序和文档的源码分别存放在 Google Code， GitHub， Java.net。 如果没有特殊声明，下面程序都采用 Apache License, Version 2.0 发布。所有文档都采用 GNU 自由文档许可证, Version 1.3 发布。

  • gaixie : 所有 Maven 结构的项目的根。我所有的 Maven 项目都需要继承它，最后发布到 Maven 中心仓库。
  • gaixie-org : Gaixie.ORG 网站静态页面，定期以 git pull 方式自动进行网站更新。
  • jibu : 一个基于 Guice 和 ExtJS 的超轻量级开源框架，已经成功应用到多个项目，可以从 这里下载 最新的编译好的打包文件。
  • jibu-doc : 以 Docbook 格式编写的 Jibu 项目文档源码。
  • bitorb : 我自己的 Linux 和 Mac OS X 的相关配置文件及脚本，主要是为了重装机器方便。
  • bitorb-org : 我个人主页的源码，用 Emacs Orgmode 编写，导出后以 rsync 方式不定期手动更新。
