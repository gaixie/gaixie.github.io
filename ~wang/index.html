程序与文档

这些年写了不少乱七八糟的程序，回过头一看基本都没什么用，水平也不高，所以基本上是边写边删，完全是为了满足自己写时的一时之快。 留下的程序和文档都散落在 Google Code， GitHub， Java.net 等几个代码库，基本上都采用 Apache License, Version 2.0 或 GNU 自由文档许可证, Version 1.3 发布。下面是我偶尔还维护一下的项目。

  • gaixie-pom : 所有 Maven 结构的项目的根。我所有开源的 Maven 项目都需要继承它，最后发布到 Maven 中心仓库。
  • bitorb : 我自己的 Linux 和 Mac OS X 的相关配置文件及脚本，主要是为了重装机器方便。
  • bitorb-org : 我个人主页的源码，用 Emacs Orgmode 编写，导出的静态页面在 gaixie.github.io 中版本化。
  • gaixie.github.io : Gaixie.ORG 网站静态页面，含我的个人主页静态页面，通过 github 的 pages 功能展示。
