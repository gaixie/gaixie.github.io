1 Debian 下通过 rbenv 和 ruby-build 安装不同版本的 ruby

先安装编译 ruby 需要的依赖包，以及 clone rbenv 和 ruby-build 源码。 

$ sudo apt-get update
$ sudo apt-get install autoconf bison build-essential \
libssl-dev libyaml-dev libreadline6-dev zlib1g-dev \
libncurses5-dev libffi-dev libgdbm3 libgdbm-dev

$ git clone https://github.com/rbenv/rbenv.git ~/.rbenv

环境变量的设置，通过加载 environment 文件，具体设置方法参考重装系统时的设置。 

$ type rbenv

$ git clone https://github.com/rbenv/ruby-build.git ~/.rbenv/plugins/ruby-build

安装 ruby 并使用 gem 

$ rbenv install -l
$ rbenv install 2.5.0
$ rbenv global 2.5.0
$ ruby -v

$ gem install bundler

参考文档：

  • https://www.digitalocean.com/community/tutorials/how-to-install-ruby-on-rails-with-rbenv-on-debian-8
  • https://github.com/rbenv/rbenv

2 OSX 通过 rbenv 和 ruby-build 安装不同版本的 ruby

通过 homebrew 安装 rbenv 和 ruby-build。 

$ brew install rbenv ruby-build

环境变量的设置，通过加载 environment 文件，具体设置方法参考重装系统时的设置。 

$ rbenv install -l
$ rbenv install 2.5.0
$ rbenv global 2.5.0
$ ruby -v

$ gem install bundler
